# 🧠 Exploratory Data Analysis (EDA) using Pandas, Matplotlib & Seaborn

This project walks through a beginner-friendly step-by-step **Exploratory Data Analysis (EDA)** workflow using Python libraries like **Pandas**, **Matplotlib**, and **Seaborn**. It is designed to help newcomers understand and visualize data effectively before building machine learning models.

---

## 📁 Dataset

Replace `'iris.csv'` with your own dataset in the code or upload a sample dataset in the repository to try it out.

---

## 📌 Key Steps Covered

1. **Data Loading & Overview**  
   - Reading CSV file using `pandas.read_csv()`  
   - Checking structure with `.head()`, `.info()`, `.describe()`

2. **Handling Missing Values**  
   - Detecting missing data  
   - Filling with mean values or dropping rows

3. **Visualizations**  
   - Histogram (`sns.histplot`)  
   - Boxplot (`sns.boxplot`)  
   - Countplot (`sns.countplot`)

4. **Correlation Analysis**  
   - Heatmap using `sns.heatmap()` to visualize correlations

5. **Outlier Detection**  
   - Filtering outliers using percentile thresholds

6. **Pair Plot**  
   - Pairwise relationship visualization using `sns.pairplot()`
